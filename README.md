# Markdown-Music-Notation (MMN)

---

MMN이란?  

마비노기, 아키에이지 등에 쓰이고 있는 MML의 불편한 점을 개선하고 마치 Markdown처럼 코드와 악보간의 괴리감을 줄이는 것을 목표로 하여 디자인되었습니다.  
다음 페이지는 MMN의 문법을 파트별로 나누어 설명하고 있습니다. 코드를 직접 수정해보세요.
